# Corrections des exercices

Les corrections sont chiffrées pour préserver l'apprentissage. 

Pour accéder aux corrections :
1. Soumettez votre travail (voir COMMENT-ACCEDER-AUX-CORRECTIONS.md à la racine)
2. Recevez le mot de passe de votre enseignant
3. Utilisez le script de déchiffrement : `../../scripts/decrypt.sh`

Les fichiers `.encrypted` contiennent les corrections détaillées.
