# Phase 6 : Déploiement de Loki

## Objectif de la phase

Configurer Loki pour l'ingestion et l'indexation des logs applicatifs. Cette phase fait partie de la Partie 3 du sujet d'examen (40 points).

## Tâches du projet

### Étape 1 : Déploiement Loki

Déployez Loki avec Docker ou Ansible.

### Étape 2 : Configuration Promtail

Configurez Promtail pour collecter les logs.

### Étape 3 : Labels

Configurez les labels pour indexer les logs.

### Étape 4 : Validation

Vérifiez que Loki ingère les logs.

## Livrable de la phase

- [ ] Loki déployé et fonctionnel
- [ ] Promtail configuré
- [ ] Logs ingérés et consultables

## Prochaine phase

Passez à la **Phase 7 : Création de dashboards Grafana**.
