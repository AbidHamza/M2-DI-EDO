# Phase 9 : Multi-environnements

## Objectif de la phase

Gérer les déploiements vers plusieurs environnements (dev, staging, production). Cette phase fait partie de la Partie 3 du sujet d'examen (40 points).

## Tâches du projet

### Étape 1 : Configuration des environnements

Configurez :
- Environnement de développement
- Environnement de staging
- Environnement de production

### Étape 2 : Pipeline multi-environnements

Adaptez le pipeline pour :
- Déployer selon la branche
- Gérer les approbations
- Utiliser les bonnes variables

### Étape 3 : Validation

Testez les déploiements vers chaque environnement.

## Livrable de la phase

- [ ] Pipeline multi-environnements fonctionnel
- [ ] Déploiements testés
- [ ] Documentation des environnements

## Prochaine phase

Passez à la **Phase 10 : Analyse et amélioration**.
