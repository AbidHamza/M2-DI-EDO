# Phase 4 : Intégration de SonarQube

## Objectif de la phase

Intégrer SonarQube dans le pipeline pour l'analyse de code. Cette phase fait partie de la Partie 3 du sujet d'examen (40 points).

## Tâches du projet

### Étape 1 : Déploiement SonarQube

Déployez SonarQube (local ou SonarCloud).

### Étape 2 : Configuration du projet

Créez un projet dans SonarQube.

### Étape 3 : Intégration dans le pipeline

Ajoutez :
- Job SonarQube dans le pipeline
- Configuration du scanner
- Quality Gate

### Étape 4 : Validation

Testez l'analyse et vérifiez les rapports.

## Livrable de la phase

- [ ] SonarQube intégré au pipeline
- [ ] Analyse fonctionnelle
- [ ] Quality Gate configuré

## Prochaine phase

Passez à la **Phase 5 : Provisionnement avec Terraform**.
